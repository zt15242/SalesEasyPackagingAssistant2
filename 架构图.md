## 数据流程

### 登录流程

```
┌──────────┐
│  User    │
│  用户     │
└────┬─────┘
     │ 1. 输入用户名密码
     ▼
┌──────────────────┐
│  Login Dialog    │
│  登录对话框        │
└────┬─────────────┘
     │ 2. 调用 login()
     ▼
┌──────────────────┐
│ ServiceFactory   │
│ getLoginService()│
└────┬─────────────┘
     │ 3. 返回服务实例
     ▼
┌──────────────────┐
│ ApiLoginService  │
│   或              │
│ MockLoginService │
└────┬─────────────┘
     │ 4. 执行登录逻辑
     ▼
┌──────────────────┐
│   HttpClient     │
│  postForm()      │
└────┬─────────────┘
     │ 5. POST /api/auth/login
     ▼
┌──────────────────┐
│  Backend API     │
│  返回 token       │
└────┬─────────────┘
     │ 6. 保存 token
     ▼
┌──────────────────┐
│   HttpClient     │
│  setAuthToken()  │
└────┬─────────────┘
     │ 7. GET /api/auth/me
     ▼
┌──────────────────┐
│  Backend API     │
│  返回用户信息      │
└────┬─────────────┘
     │ 8. 返回 LoginResult
     ▼
┌──────────────────┐
│  Login Dialog    │
│  显示登录结果      │
└──────────────────┘
```

## 组件职责

### 1. ApiConfig
- ✅ 存储API基础URL
- ✅ 定义所有API端点
- ✅ 提供URL构建方法
- ✅ 支持动态修改域名

### 2. HttpClient
- ✅ 封装HTTP请求
- ✅ 管理认证Token
- ✅ 统一错误处理
- ✅ JSON序列化/反序列化

### 3. ServiceFactory
- ✅ 创建服务实例
- ✅ 管理服务生命周期
- ✅ 支持Mock/Real切换

### 4. LoginService
- ✅ 用户登录
- ✅ 用户登出
- ✅ 检查登录状态
- ✅ 获取当前用户

### 5. NotificationService
- ✅ 获取通知列表
- ✅ 标记已读
- ✅ 删除通知
- ✅ 获取未读数量

## 设计模式

### 1. 工厂模式 (Factory Pattern)
```
ServiceFactory
  ├── getLoginService()
  └── getNotificationService()
```

### 2. 策略模式 (Strategy Pattern)
```
LoginService (接口)
  ├── ApiLoginService (真实实现)
  └── MockLoginService (模拟实现)
```

### 3. 单例模式 (Singleton Pattern)
```
object ApiConfig { ... }
object HttpClient { ... }
object ServiceFactory { ... }
```

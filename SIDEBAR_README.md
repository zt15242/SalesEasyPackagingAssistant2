# SalesEasy 侧边栏工具窗口

## 功能概述

SalesEasy 侧边栏提供了用户登录和通知中心两大核心功能,方便用户在 IntelliJ IDEA 中直接管理账户和查看系统通知。

## 界面布局

### 1. 用户登录区域

#### 未登录状态
- **用户名输入框**: 输入您的用户名
- **密码输入框**: 输入您的密码
- **登录按钮**: 点击进行登录验证

#### 已登录状态
- **用户头像**: 显示用户图标
- **用户名**: 显示当前登录的用户名(加粗显示)
- **登录状态**: 显示"已登录"状态,带绿色对勾图标
- **登出按钮**: 点击退出登录

### 2. 通知中心区域

- **通知列表**: 
  - 每条通知包含:
    - 图标(信息/成功/警告/错误)
    - 标题(加粗)
    - 消息内容
    - 时间戳
  - 支持滚动查看
  - 最多保留 50 条通知

- **操作按钮**:
  - **刷新按钮**: 刷新通知列表
  - **清空按钮**: 清空所有通知

## 通知类型

- **INFO** (信息): 蓝色信息图标
- **SUCCESS** (成功): 绿色对勾图标
- **WARNING** (警告): 黄色警告图标
- **ERROR** (错误): 红色错误图标

## 使用方法

### 打开侧边栏
1. 在 IntelliJ IDEA 中,点击右侧边栏的 "SalesEasy Assistant" 标签
2. 侧边栏会自动展开显示

### 登录流程
1. 在"用户登录"区域输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后,界面会切换到用户信息显示
4. 通知中心会收到登录成功的通知

### 登出流程
1. 点击"登出"按钮
2. 确认登出操作
3. 界面切换回登录表单

### 查看通知
1. 通知列表会自动显示最新的通知
2. 点击"刷新"按钮可以手动刷新通知
3. 点击"清空"按钮可以清空所有通知

## 技术实现

### 核心类
- `SalesEasyToolWindowFactory`: 工具窗口工厂类,负责创建工具窗口
- `SalesEasyToolWindowPanel`: 主面板类,包含所有 UI 组件和业务逻辑
- `NotificationItem`: 通知数据类
- `NotificationType`: 通知类型枚举
- `NotificationCellRenderer`: 通知列表渲染器

### 注册配置
在 `plugin.xml` 中注册工具窗口:
```xml
<toolWindow id="SalesEasy Assistant" 
            secondary="false" 
            icon="AllIcons.General.Settings" 
            anchor="right" 
            factoryClass="top.javaxiaowu.saleseasypackagingassistant.ui.SalesEasyToolWindowFactory"/>
```
